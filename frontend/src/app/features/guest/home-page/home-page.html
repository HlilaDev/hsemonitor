Une plateforme conçue pour le HSE industriel moderne

HSE Monitor combine la gestion applicative SaaS, la supervision terrain, l’IoT et l’intelligence artificielle dans une seule interface web. La plateforme aide les entreprises à réduire les risques, améliorer la conformité et prendre des décisions rapides à partir de données réelles.

SaaS Architecture multi-entreprise avec gestion centralisée par super admin.

IA Vision Détection de port de casque, gilet, comportements non conformes et anomalies.
Web Interface moderne pour le monitoring, les alertes, les formations et les rapports.

Comment fonctionne la plateforme

De l’installation des devices jusqu’à la visualisation sur le dashboard, tout le workflow HSE est centralisé.

1

Création de la company

Le super admin crée une nouvelle entreprise et assigne son administrateur principal.

2

Installation des devices

L’admin d’entreprise configure les Raspberry, ESP, capteurs et les affecte aux zones.

3

Collecte des données

Les devices envoient les données terrain et les alertes via MQTT au serveur backend.

4

Visualisation et action

Le dashboard web affiche les mesures, les incidents, les trainings et les alertes IA en temps réel.
